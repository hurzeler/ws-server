Whenever PRESET mode is selected the Values are sent to WBU as below

Adjusted "Minus" Value 	-	WTRTUv##     
Adjusted "Set" Value 	-	WTRLAv##
Adjusted "Plus" Value	-	WTRTUv##

where ## is the new value


For Throttle and Regen mode, no notifications are sent to WBU.

All defaults are coded in "index.html", but are then stored in browser storage and are subsequently used from there. Any edits are updated to the browser storage.
